## Claude Code responsibilities

Claude Code is primarily used for:

- feature implementation
- architecture changes
- refactoring
- multi-file modifications
- test creation
- documentation updates

## Required behavior

1. Inspect the repository first.
2. Create a concise plan before substantial work.
3. Keep changes focused on the requested task.
4. Ask before making destructive, security-sensitive, or irreversible changes.
5. Do not commit or push unless explicitly requested.
6. Run the relevant repository checks after editing.
7. Review the final diff before declaring completion.
8. Report exactly what changed and what remains unresolved.

## Local files

Do not commit local-only Claude settings, credentials, caches, or machine-specific configuration.
